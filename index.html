Simple Calculator App

This is a minimalist single-page web application that provides a basic calculator functionality.

Features

  • Addition, Subtraction, Multiplication, Division
  • Clear button to reset the calculator
  • Decimal point support
  • Basic error handling (e.g., division by zero, invalid input)
  • Responsive design for various screen sizes

How to Use

  1. Enter the first number using the number buttons.
  2. Click an operator (+, -, *, /).
  3. Enter the second number.
  4. Click the equals (=) button to see the result.
  5. Use the 'C' button to clear the current input or reset the calculator.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript (ES6+)
