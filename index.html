Simple Calculator App

A basic calculator web application built with HTML, CSS, and JavaScript. It provides standard arithmetic operations in a clean, user-friendly interface.

Features

  • Addition, Subtraction, Multiplication, Division
  • Percentage calculation (e.g., 100 + 10% = 110, 50% = 0.5)
  • Sign change (+/-)
  • Decimal point support
  • Clear functionality ('C')
  • Visually clean and professional design

How to Use

Click the number buttons to input values, then select an operator (+, -, *, /). Click '=' to see the result. Use 'C' to clear the current input and reset the calculator. The '+/-' button changes the sign of the current number, and '%' performs percentage calculations.
